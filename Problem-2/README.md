### 问题描述

给定一个整数数组 `nums` 和目标数字 `target`， 请你在数组中找到和为目标值的两个整数，并返回他们的下标。输入中存在负数。

可以假设每种输入只会对应一个答案。但是，数组中的同一个元素在答案中不会重复出现

返回值可以按任意顺序。

##### 范例其一

输入：nums = [2, 7, 11, 15], target = 9

返回值： [0, 1]

##### 解法其一

遍历数组。固定一个值，在数组中遍历（数组并非升序或降序）寻找和为目标值的另一个值。

##### 解法其二

创建一个长度为 `target` 的哈希表 `hashTable`。 遍历数组，对于数组中每一个元素，检查 `target - element` 是否存在。存在就获取相应的元素下标，不存在就创建。
