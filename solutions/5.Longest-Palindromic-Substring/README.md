# 5. Longest Palindromic Substring

### 题目描述

给定一个字符串，要求这个字符串当中最长的回文串。

**示例1:**

```
Input: "babad"
Output: "bab"
Note: "aba" is also a valid answer.
```

**示例2:**
```
Input: "cbbd"
Output: "bb"
```

### 题目解析

参考马拉车算法：https://en.wikipedia.org/wiki/Longest_palindromic_substring#Manacher's_algorithm
